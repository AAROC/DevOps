ansible-role-umd3
=================

# Ansible Galaxy role for installing Unified Middleware Distribution (version 3)

Installs the EPEL repository (Extra Packages for Enterprise Linux) for RHEL/CentOS 6.x.

## Requirements

None.

## Role Variables

None.

## Dependencies

None.

## Example Playbook

    - hosts: servers
      roles:
        - { role: brucellino.repo-umd3 }
